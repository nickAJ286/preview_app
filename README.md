# アプリケーション概要
学習内容の投稿を行い、復習として投稿から翌日、そこから3日後（学習から4日後）、さらに一週間後（学習から11日後）、さらに1カ月後（学習から41日後）の計4回投稿内容が登録アドレスに送られる
